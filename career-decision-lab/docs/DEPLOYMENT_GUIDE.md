## 🎯 如何使用

### 本地开发

#### 1. 启动项目

```bash
# 进入项目目录
cd "d:\code\carrer select\career-decision-lab"

# 启动开发服务器
npm run dev

# 浏览器访问
http://localhost:3000
```

#### 2. 访问新页面

**新问卷页面**：
```
http://localhost:3000/questionnaire-v5
```

**新结果页面**：
```
http://localhost:3000/results-v5
```

**API测试**：
```
http://localhost:3000/api/ai/generate
```

#### 3. 测试流程

1. **从首页开始**
   ```
   http://localhost:3000
   ```

2. **填写基础信息**（/info页面）
   - 年级、专业、考研计划等

3. **开始问卷**（会自动跳转到 /questionnaire-v5）
   - 完成5个section的47道题目
   - 预计8-12分钟

4. **查看结果**（自动跳转到 /results-v5）
   - AI生成个性化分析
   - 查看匹配的职业路径
   - 查看能力结构
   - 获取行动建议

## 🚀 部署指南

### Netlify部署

1. **推送代码到Git**
   ```bash
   git add .
   git commit -m "feat: complete v5.0 refactoring"
   git push origin main
   ```

2. **在Netlify配置环境变量**
   - 登录 Netlify
   - 进入 Site settings → Environment variables
   - 添加：`GLM_API_KEY` = `你的API_Key`
   - 重新部署

### Vercel部署

1. **推送代码到Git**
   ```bash
   git add .
   git commit -m "feat: complete v5.0 refactoring"
   git push origin main
   ```

2. **在Vercel配置环境变量**
   - 登录 Vercel
   - 进入 Project Settings → Environment Variables
   - 添加：`GLM_API_KEY` = `你的API_Key`
   - 重新部署

### 本地生产构建

```bash

npm run build

# 启动生产服务器

http://localhost:3000
```
